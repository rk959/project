# 🎵 Music Mood Suggester

## 📌 Project Overview

The **Music Mood Suggester** is a simple Python project that recommends songs based on the user's mood.
The program asks the user to enter their mood and then suggests songs that match that mood.

This project demonstrates basic **Python programming, data handling with pandas, and conditional logic**.

---

## 🚀 Features

* Suggests songs based on user mood
* Uses a dataset structure with pandas DataFrame
* Randomly recommends a song from the selected mood
* Beginner-friendly Python project

---

## ▶ How to Run the Project

1. Install Python (3.x recommended)

2. Install required libraries

```
pip install pandas
```

3. Run the program

```
python music_mood_suggester.py
```

4. Enter your mood when prompted

Example moods:

* happy
* sad
* relax
* energetic
